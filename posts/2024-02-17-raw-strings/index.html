Let‚Äôs talk about strings

In R, strings can contain " and ' characters, the rules are simple. If you are inside a " string, you can have a ' character, if you are inside a ' string, you can have a " character without needing to escape them:

"hello 'world'"
[1] "hello 'world'"
'bonjour "monde"'
[1] "bonjour \"monde\""

And otherwise you need to escape them

"ciao \"mondo\""
[1] "ciao \"mondo\""
'hola \'mundo\''
[1] "hola 'mundo'"

raw strings attached

ü§¢ escaping strings is a pain, so R 4.0.0 included raw strings, which you can learn more about in this article from Josiah Parry. Bonus points for the Seinfeld gif.
